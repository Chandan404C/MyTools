How to Use

  1. Upload – Drop or select any audio file (MP3, WAV, OGG, M4A, FLAC…).
  2. Set trim points – Drag the handles on the range slider or type exact times in the Start / End fields.
  3. Preview – Hit Play to hear the full track, or Preview Trim to hear only the selected region.
  4. Download – Click Trim & Download and choose your output format.

Everything is processed locally — your audio never leaves your device.

Frequently Asked Questions

✂️ What audio formats does it support?
You can upload MP3, WAV, OGG, M4A, FLAC, AAC, WebM and any other format your browser can decode. WAV is the most universally supported output format.
🔒 Is my audio file kept private?
Yes. The tool uses the browser's built-in Web Audio API. Your file is never uploaded to any server — it stays entirely on your device.
📏 Is there a file size limit?
There is no imposed limit. Very large files are limited only by your device's available memory and browser capability.
🎵 Can I preview before downloading?
Yes! Click Preview Trim or the scissors button to play only the selected region before you download it.
⏱️ How precise can my trim points be?
You can enter start and end times accurate to the millisecond using the text input fields below the waveform.
📱 Does it work on mobile?
Yes! The interface is fully responsive. The Web Audio API is supported on all modern mobile browsers including Chrome and Safari on iOS.
