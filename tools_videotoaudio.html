Frequently Asked Questions

🎬 What video formats does it support?
You can upload MP4, WebM, OGG, MOV, AVI, MKV and any other format your browser can decode. MP4 and WebM are most widely supported.
🔒 Is my video file kept private?
Yes. The tool uses the browser's built-in Web Audio API and HTMLMediaElement. Your file is never uploaded to any server — it stays entirely on your device.
📏 Is there a file size limit?
There is no imposed limit. Very large files are limited only by your device's available memory and browser capability.
🎵 Can I preview the audio before downloading?
Yes! Click Preview Audio to play just the audio track extracted from the video before you download it.
🎧 What audio formats can I export to?
You can export to WAV (lossless, universally supported), and browser-dependent formats like MP3, OGG and WebM audio.
📱 Does it work on mobile?
Yes! The interface is fully responsive. The Web Audio API is supported on all modern mobile browsers including Chrome and Safari on iOS.
